# Lymphoma Histopathology Classification

Deep learning classification of lymphoma subtypes (CLL, FL, MCL) from H&E-stained histopathology patches. B

## Dataset

[Kaggle Malignant Lymphoma Classification](https://www.kaggle.com/datasets/andrewmvd/malignant-lymphoma-classification) — 374 H&E-stained patches across 3 classes.
